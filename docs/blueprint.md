# **App Name**: The Scriptorium Cypher: The Sentient Codex

## Core Features:

- In-Situ Manifestation: Invokes the Scribe on a specific element, dissolving the UI around that element and manifesting a temporary Cognitive Canvas in-situ.
- Ambient Knowledge Streams: Visualizes knowledge pathways as ambient Aetheric Streams flowing subtly in the background of the OS.
- Architect's Annotations: Architects can annotate any Sigil; these annotations appear as personal, glowing glyphs that only they can see.
- Real-Time Simulation: Code blocks are now full-fledged, embedded sandboxes. An architect can modify the code within the Sigil and see the visual output of the component render in real-time, directly on the canvas.
- Forking the Scripture: Architects can 'fork' the Living Sigil. This creates a new, community-visible branch of knowledge.
- On-Demand Canvas Generation: The Scribe Agent is a direct conversational interface that analyzes the query and generates a brand new Living Sigil on the fly, synthesizing all the relevant information into a single, unified, and perfectly tailored explanation tool.
- Deep, Bidirectional Linking: Every element within a Living Sigil is now a Forge Link: Clicking a Forge Link in the Codex instantly teleports the architect to the exact line of code or node in their Loom Studio workspace, and vice versa.

## Style Guidelines:

- Primary color: Luminous light-purple (#9400D3) evokes futuristic exploration.
- Background color: Near-black (#08080B) supports contrast and helps imply infinite space.
- Accent color: Pale-yellow (#FFFF8F) as the highlight color of Aetheric threads
- For Headings, Logos, and Sacred Names: Eurostile Unicase Bold Extended (customized) or Monument Extended. Feels monumental, engineered, secret.
- For All UI Text and Body Copy: Söhne (or Inter). Neutral, timeless, dissolves into interface
- For Monospace, Code, and Data: Recursive Mono (or Geist Mono). Morphing clarity; changes with user state
- Use glowing glyphs and minimalist icons to represent concepts and actions.
- Design Cognitive Canvases as dynamic, free-form spaces, avoiding linear structures and promoting visual-first presentation.
- Implement subtle, ethereal animations for transitions and interactions within the Constellation Map and Cognitive Canvases.
